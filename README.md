# Xbox Cloud iPhone - GitHub Pages

Déploiement rapide sur GitHub Pages.

## Fichiers
- `index.html`
- `manifest.json`
- `assets/icon.svg`
- `assets/splash.svg`

## Déploiement
1. Crée un nouveau repo GitHub, par exemple `xbox-cloud-iphone`.
2. Envoie tous les fichiers de ce dossier à la racine du repo.
3. Dans GitHub, va dans **Settings > Pages**.
4. Sous **Build and deployment**, choisis **Deploy from a branch**.
5. Sélectionne la branche **main** et le dossier **/(root)**.
6. Sauvegarde.
7. Attends 1 à 3 minutes.
8. Ouvre l’URL fournie par GitHub Pages, par exemple `https://TON-USERNAME.github.io/xbox-cloud-iphone/`.
9. Ouvre cette URL dans Comet sur iPhone.

## Utilisation iPhone
- Ouvre l’URL GitHub Pages dans Comet.
- Appuie sur **Lancer Xbox Cloud**.
- Si tu veux un accès rapide, ajoute ensuite la page à l’écran d’accueil.

## Note
GitHub Pages sert le site en HTTPS, ce qui marche mieux pour une web app iPhone qu’un fichier HTML local ouvert depuis l’app Fichiers.
